PDF Reader Web Application

Project Overview

Architected and delivered a cutting-edge, full-stack web platform enabling users to securely upload, semantically search, and interact with PDF documents through an AI-powered conversational interface. Leveraged state-of-the-art technologies to provide real-time, context-aware insights from user documents, driving productivity and knowledge discovery.

Technology Stack

  • Frontend: Next.js, React 19, Tailwind CSS, Clerk (authentication)
  • Backend: Node.js, Express, LangChain, OpenAI API, Qdrant (vector database), BullMQ (distributed background processing)
  • Other: Multer (file uploads), Docker (containerization), Redis (queue management)

Key Achievements & Features

  • Engineered secure, scalable user authentication and per-user document isolation using Clerk and Express middleware.
  • Implemented asynchronous, distributed PDF processing pipeline with BullMQ and Redis, enabling high-throughput document ingestion and embedding.
  • Integrated OpenAI's advanced language models and LangChain for semantic search, enabling users to query documents in natural language and receive contextually relevant, referenced answers.
  • Designed a modern, responsive UI/UX with real-time chat, document upload, and source citation features, enhancing user engagement and accessibility.
  • Ensured robust security by enforcing strict access controls, secure file handling, and token-based API authentication.

Challenges & Solutions

  • Challenge: Efficiently processing and embedding large PDF files without blocking user interactions.
    Solution: Designed a background job system using BullMQ and Redis, decoupling heavy processing from the main application and ensuring seamless user experience.
  • Challenge: Delivering accurate, context-rich answers from unstructured PDF data.
    Solution: Leveraged LangChain and OpenAI embeddings to chunk, vectorize, and retrieve relevant document sections, powering precise semantic search and chat responses.
  • Challenge: Maintaining strict data privacy and per-user access control.
    Solution: Implemented per-user storage directories and authentication middleware, ensuring users can only access their own documents and data.
